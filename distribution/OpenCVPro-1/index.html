OpenCVPro

A library by Greg Borenstein for the Processing programming environment.
Last update, 06/24/2013.

Computer vision with OpenCV based on the official OpenCV Java API.

Download

Download OpenCVPro version 0.2.1 (1) in .zip format.

Installation

Unzip and put the extracted OpenCVPro folder into the libraries folder of your Processing sketches. Reference and examples are included in the OpenCVPro folder.
